# RiskOfImpact

A fun collection on items that will be expanded on moving forward.
Compatible with LookingGlass (and some items appear on CelestialWarTank)

## Content

Adds **6 items** and **1 equipment**:

- **1 Uncommon**
- **2 Legendary**
- **1 Void**
- **2 Lunar**
- **1 Lunar Equipment**
